# Image Directory Structure

This directory contains all images for the Warhammer 40K website.

## Folder Structure

### `/characters`
Store character portrait images here (150x200px recommended)
- `emperor.png` - The God-Emperor of Mankind
- `guilliman.png` - Roboute Guilliman
- `abaddon.png` - Abaddon the Despoiler
- `corvus.png` - Corvus Corax
- `eldrad.png` - Eldrad Ulthran
- `ghazghkull.png` - Ghazghkull Mag Uruk Thraka
- `szarekhan.png` - Szarekhan Dynasty
- `omnissiah.png` - The Omnissiah

### `/factions`
Store faction leader and representative images here (200x300px recommended)
- `imperium.png` - Imperium of Man Leader
- `chaos.png` - Chaos Forces Leader
- `aeldari.png` - Aeldari Leader
- `orks.png` - Orks Leader
- `necrons.png` - Necrons Leader
- `tau.png` - Tau Empire Leader

### `/gallery`
Store hero gallery images for the home page here (800x300px recommended)
- `emperor-throne.png` - The God-Emperor
- `imperium.png` - The Imperium
- `golden-throne.png` - The Golden Throne

## Image Format Recommendations

- **Format**: PNG or JPG
- **Character Portraits**: 150x200px (3:4 aspect ratio)
- **Faction Leaders**: 200x300px (2:3 aspect ratio)
- **Hero Gallery**: 800x300px (8:3 aspect ratio)
- **Optimize**: Compress images to reduce file size

## How to Add Images

1. Save your image file in the appropriate folder
2. Make sure the filename matches the names referenced in the HTML files
3. Refresh your browser to see the updated images

## Current Setup

All HTML files have been updated to use local image paths. Simply add your images to these directories and they will appear on the website automatically.
