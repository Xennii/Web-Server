## How to Add Images

1. Navigate to the appropriate folder in `/static/images/`
2. Save your image with the exact filename listed above
3. Refresh your browser to see the image appear on the website

## Supported Formats

- PNG (recommended for transparency and quality)
- JPG (recommended for photographs and compressed files)
- WebP (for modern browsers and smaller file sizes)

## Image Optimization Tips

- Compress images using tools like TinyPNG or ImageOptim
- Use appropriate dimensions (listed above) for best results
- Keep file sizes under 500KB for fast loading
